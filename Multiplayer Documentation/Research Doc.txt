===Key Names; research.json===
// Most keys are optional, asterisk denotes mandatory, double asterisk denotes conditionally mandatory (read description) //
// C = compare; S = string; I = integer; F = float; D = double; B = boolean; V = varies; N = null //
// Quotes are included: compare: "old:new"; string: "value"; integer: whole number; float: decimal //
// double: whole/decimal; boolean: false/true; varies: depends on parent; null: no value; IB = integer boolean: 0=false/1=true //
// For keys that accept B or IB, removing the key is equivalent to false //
disabledWhen I          // 1 = Tanks, unused; 2 = Cyborgs; 4 = VTOLs; 8 = Satellite Uplink Center; 16 = Laser Satellite Command Post //
iconID S                // Icon shown on topic, find ==iconID== below for a list //
id* S                   // Needs to be same as the JS object name, and needs to be unique //
msgName S               // Message ID of message in Intelligence Display, see "messages" folder, Mutual Exclusion: techCode //
keyTopic IB             // Unused in multiplayer/skirmish //
imdName S               // File name for 3D model, find ===3D Models=== below for a list, Mutual Exclusion: statID //
name* S                 // Display name //
replacedComponents SC   // Components to be replaced //
redComponents S         // Components to be made redundant without replacing //
requiredResearch S      // Research IDs of required research //
researchPoints D        // How many research points it takes to research, max is 65535 (16-bit) //
researchPower D         // How much power it takes to research, max is 65535 (16-bit) //
resultStructures S      // Structures to be made available //
resultComponents S      // Components to be made available //
results> N              // Upgrades to be given, details follow //
>class* S               // Body; Brain; Building; Construct; ECM; Propulsion; Repair; Sensor; Weapon //
>filterParameter S      // The stat parameter to filter for, find ==Filter Parameters== below for a list of useful ones //
>filterValue V          // The stat value to filter for, find ===List of weapon classes=== below for a list of weapon classes //
>parameter* S           // The parameter to be modified //
>value* V               // The amount the parameter is modified by, can sometimes be negative //
statID S                // The 3D component to be displayed in the research topic, Mutual Exclusion: imdName //
subgroupIconID S        // Icon shown on topic, find ==subgroupIconID== below for a list //
techCode** IB           // No Intelligence Display entry, mandatory if msgName is not provided, Mutual Exclusion: msgName //

===Parameter Names===
// Special note for 'HitPointPct', it increases the total HP, it is 100% by default, giving +25% would buff final HP by 25% //
// This is additive, +25% to weapon, +25% to body, and +25% to propulsion would be 175% (+75%) total.
